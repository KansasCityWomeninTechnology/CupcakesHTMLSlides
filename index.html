DEPLOYING

Right now, your website is only on your computer.

To let other people see it, we need to "push" it to GitHub.

DEPLOYING

  • Open an SSH terminal in Codeanywhere by right-clicking on codingandcupcakes in the left sidebar and selecting SSH Terminal. When the terminal window opens, type the following bolded Git commands:
    • git status - shows changes you've made to files on your local computer
    • git add --all - adds files to the batch, kind of like adding ingredients to cupcake batter
    • git status - the changes should show in green now instead of red
    • git commit -m "{your message here}" - commits files with a message about what you changed
    • git push origin master - pushes the changes to your branch in GitHub
    • git push origin master:gh-pages - creates a page where your repository is hosted.
